切绳子

题目大意

从N条长度为L_i的绳子中切出k条长度相同的短绳子，问短绳子的最长长度。

0 < Li <= 100000.00

0 < n <= 10000

0 < k <= 10000

解法

一个很裸的二分答案。因为单调性显然，所以二分答案k，然后O(N)判断当前答案是否可行即可。这题没什么可说的，唯一想说的就是，因为我太菜了，所以我一开始写的是double二分，然后精度被卡飞了，有两个点一直WA。后来才发现保留两位小数其实可以读入的时候全部乘100，输出的时候再除以100.0，然后就是整数二分了。

我好菜啊。。我什么都不会啊。。
